# Lava autotest

Perform testing all in one.

```
usage: run.py [-h] [-i INPUT] [-d DIR] [-l LOCATION] [-c]

An auto-test tool for lava project.

optional arguments:
  -h, --help            show this help message and exit
  -i INPUT, --input INPUT
                        specify input SysY source file, default to empty, that means run files in script configuration
  -d DIR, --dir DIR     specify input SysY source files directory, default to ./cases
  -l LOCATION, --location LOCATION
                        specify location of lacc
  -c, --cross           cross-compile mode
```
